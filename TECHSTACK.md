# Tech Stack

## Core Framework

- Next.js 16.2.2 (App Router)
- React 19.2.4
- TypeScript 5

## Styling

- Tailwind CSS 4
- Lucide React icons

## Database

- PostgreSQL
- Prisma 6.19.3
- Prisma Client for type-safe queries

## Authentication

- jose for JWT signing and verification
- bcryptjs for password hashing
- Cookie-based session storage via Next.js cookies

## API & Server

- Next.js Route Handlers for server-side APIs
- Built-in middleware for route protection
- File upload API route for image handling

## Developer Tools

- ESLint
- dotenv for environment configuration
- tsx for running TypeScript scripts
